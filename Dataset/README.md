## Data Dictionary

| Name | Description | Type | Unit |
|---|---|---|---|
| `date` | Date and time | string | date, hour, min, sec |
| `t` | Time since start of the dataset | integer | hour |
| `yTi1` | Temperature 1 (Room 1 indoor temperature) | continuous | °C |
| `yTi2` | Temperature 2 (Room 2 indoor temperature) | continuous | °C |
| `yTi3` | Temperature 3 (Room 3 indoor temperature) | continuous | °C |
| `yTi4` | Temperature 4 (Room 4 indoor temperature) | continuous | °C |
| `Ta` | Ambient temperature | continuous | °C |
| `Gv` | Global horizontal solar radiation | continuous | W/m² |
| `Ph1` | Heating power in northern circuit | continuous | W |
| `Ph2` | Heating power in southern circuit | continuous | W |
